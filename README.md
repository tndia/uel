# UEL — GitHub Pages

- Site Jekyll (Just the Docs): `uel_plano-de-estudos/`
- Deploy: `.github/workflows/pages.yml` (GitHub Actions)

## Como publicar

1. GitHub → **Settings → Pages**
2. **Source: GitHub Actions**
3. Faça um push para `main` e acompanhe o workflow em **Actions**.

> Este setup existe porque o build padrão do GitHub Pages (branch) espera o site no root ou `/docs`.
