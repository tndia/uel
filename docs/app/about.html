Como publicar no GitHub Pages

  1. Crie um repositório (ex.: planner-uel).
  2. Envie todos os arquivos desta pasta (raiz com index.html).
  3. Em Settings → Pages, selecione Deploy from a branch e a branch main / pasta / (root).
Se você hospedar em subpasta (não recomendado), ajuste caminhos no service worker e nos links.

Backup de dados

Use Configuração → Exportar para salvar um backup em JSON e poder importar em outro navegador/computador.
