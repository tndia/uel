Estatísticas

Página: Estatísticas

Estatísticas ajudam a evitar autoengano: você vê o que está praticando de verdade.

Métricas úteis¶

  • minutos totais e sessões (Pomodoro)
  • streak (sequência) — útil como motivação, não como meta absoluta
  • evolução por matéria/módulo (domínio/revisões)

Como interpretar sem paranoia¶

  • Use tendências semanais/mensais (não o dia isolado).
  • Quando cair: ajuste carga ou disponibilidade (não “força de vontade”).

Revisão semanal recomendada (10 min)¶

  • Qual matéria está “sangrando” mais erros?
  • O plano está realista?
  • Estou revisando antes de esquecer?
  • Preciso reduzir o escopo?
