# Atalhos de teclado

## App

- **Ctrl/⌘ + K** → foco na busca rápida (quando o campo existe na página)
- **Esc** → fechar drawer/modais (quando aplicável)

> O app foi implementado com atalho Ctrl/⌘+K no shell (`app/assets/js/app.js`).

## Navegação rápida

Mesmo sem atalhos adicionais, você pode usar:

- menu lateral (desktop)
- drawer (mobile)

## Sugestão (se você for evoluir)

Se adicionar atalhos por página, documente aqui e mantenha consistência:

- `Ctrl/⌘ + Enter` para “salvar”
- `?` para abrir ajuda
