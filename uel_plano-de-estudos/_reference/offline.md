# Offline e PWA (guia rápido)

- O app registra Service Worker para cache offline.
- O cache é versionado por uma string (`CACHE`).
- Mudanças grandes → incremente `CACHE` para forçar atualização.

Guia completo: **[Modo offline]({{ "/docs/offline/" | relative_url }})**.
