# Publicação

Guia completo para publicar seu fork no **GitHub Pages**, respeitando:

- build padrão do Pages (Jekyll em safe mode),
- uso correto de `url`/`baseurl`,
- links relativos e rotas estáveis,
- SEO básico (sitemap/metadata),
- sem dependências externas frágeis.

Comece por: **[Publicar no GitHub Pages]({{ "/docs/publish/" | relative_url }})**.
