# Calendário

Página: [Calendário]({{ "/app/calendar.html" | relative_url }})

O calendário serve para visualizar:

- o que foi feito (histórico),
- o que está planejado,
- compromissos relevantes (quando você registra tarefas).

## Usos comuns

- revisar semanalmente (domingo):
  - o que rendeu,
  - o que ficou pendente,
  - ajustar a próxima semana.
- identificar “buracos” (dias sem estudo) e corrigir a agenda.

## Dica
Não use o calendário para “se culpar”. Use para **decidir o próximo ajuste**.
