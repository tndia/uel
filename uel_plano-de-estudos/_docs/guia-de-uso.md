# Guia de uso

Este guia explica o **fluxo de estudo recomendado** e como usar cada área do app no dia a dia.

## Fluxo recomendado (bem prático)

1. **Configuração**: defina prova, carga semanal e disponibilidade.
2. **Plano semanal**: o app gera uma agenda inicial (você ajusta conforme sua realidade).
3. **Revisão espaçada**: todo dia, faça primeiro o que está “vencendo”.
4. **Exercícios**: pratique o que estudou (e registre erros).
5. **Redação / Simulados**: use como treino de performance (tempo, checklist, correções).
6. **Estatísticas**: semanalmente, revise o que está evoluindo e o que precisa de reforço.

> Dica: comece simples. O app foi pensado para funcionar sem “setup perfeito”.
