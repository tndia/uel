# Módulos e conteúdo

Páginas:
- [Módulos]({{ "/app/modules.html" | relative_url }})
- [Detalhe do módulo]({{ "/app/module.html" | relative_url }})

O app organiza o estudo em **módulos**: unidades pequenas com objetivos, checklist, recursos e prática.

## Como usar “Módulos”

- Veja a lista geral por matéria.
- Marque progresso (“domínio”/mastery) conforme você avança.
- Use tags para filtrar temas.

## Como usar “Detalhe do módulo”

Um módulo normalmente traz:

- **Goals**: o que você deve dominar
- **Checklist**: itens para confirmar antes de “fechar” o tema
- **Resources**: links/leitura (sem depender de serviços externos)
- **Practice**: exercícios sugeridos
- **EstSessions**: estimativa de sessões para completar

> Dica: trate “domínio 3” como “consigo resolver/explicar sem olhar”.

## Revisão espaçada ligada a módulos

O app guarda:
- quando você estudou por último,
- quando deve revisar,
- quantas “quedas” (lapses) aconteceram.

Isso alimenta a fila de **Revisões**.
