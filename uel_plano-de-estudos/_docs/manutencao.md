# Manutenção e dados

Como o projeto é **local‑first**, seus dados ficam no navegador. Isso é ótimo para privacidade, mas exige **backup** para evitar perdas.

Nesta seção você encontra:

- como funciona o armazenamento local,
- como exportar/importar backup,
- como funciona o modo offline,
- compatibilidade entre navegadores e atualização de versão.
