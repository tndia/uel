# Redação

Página: [Redação]({{ "/app/writing.html" | relative_url }})

A redação é uma habilidade: você melhora com **volume + feedback + correção por rubrica**.

## Rotina mínima (funciona)

- 1 redação/semana nas primeiras semanas
- 2 redações/semana na reta final
- Revisão de “estrutura” a cada texto

## Estrutura sugerida (genérica)

1. Introdução: tese + recorte
2. Desenvolvimento 1: argumento + evidência
3. Desenvolvimento 2: argumento + evidência
4. Conclusão: síntese + proposta (quando aplicável)

> Ajuste ao modelo cobrado no seu curso/trilha.

## Como usar o app

- Escolha um tema (o app traz prompts)
- Escreva ou cole o texto
- Registre:
  - data,
  - autoavaliação (se quiser),
  - pontos de melhoria

## Checklist de correção (rápido)

- [ ] Atendeu ao tema e recorte?
- [ ] Tese clara?
- [ ] Argumentos sustentados?
- [ ] Coesão (conectivos)?
- [ ] Norma/pontuação adequadas?
- [ ] Conclusão consistente?
