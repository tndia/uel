# Obras

Página: [Obras]({{ "/app/literature.html" | relative_url }})

Para trilhas que exigem leitura de obras, o app organiza:

- lista de obras,
- progresso por obra,
- notas rápidas e marcos (capítulos/temas).

## Como usar

- Defina uma meta realista: capítulos por semana.
- Faça revisões curtas (resumos + personagens + temas).
- Antes de simulados: releia suas notas e faça 5 perguntas-chave por obra.

## Dica prática: “cartão de obra”

Para cada obra, mantenha:
- enredo (3 linhas),
- personagens centrais,
- temas recorrentes,
- 3 citações/frases-chave,
- 2 conexões com atualidades.

Isso acelera redação/discursiva.
