# Sobre

O **Planner UEL Pro** é um projeto de organização de estudos com princípios:

- **Local-first**: seus dados ficam com você.
- **Sem dependências externas frágeis**: app estático simples.
- **Documentação no repositório**: fácil manter e publicar no Pages.
