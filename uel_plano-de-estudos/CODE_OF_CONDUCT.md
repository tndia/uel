# Código de Conduta

Adotamos o **Contributor Covenant** como guia de convivência.

- Seja respeitoso(a) e colaborativo(a).
- Sem assédio, discriminação ou ataques pessoais.
- Em caso de conflito, priorize o diálogo e a boa-fé.

Se você presenciar comportamento inadequado, abra uma issue (ou use um canal privado definido pelo mantenedor, se houver).

> Este documento é um resumo. Se quiser, você pode substituir por uma versão completa do Contributor Covenant.
